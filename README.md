# TokyoOS

### Directories:
- bin/      : The linked binaries and disk images.
- lib/      : The compiled object files.
- log/      : Log files.
- src/      : Source code files.
- utils/    : Utilities for making the final disk image.

---

### Features implemented:
- Bootloader
- Paging
